# 📋 ClipSync

**ClipSync** is a lightweight, secure, and lightning-fast clipboard synchronization tool that bridges the gap between your **Android** and **Windows** devices. Copy text on your phone, and it's instantly available on your PC—and vice versa. No cloud, no lag, and 100% open source.

---

## 🚀 Key Features

- **⚡ Instant Peer-to-Peer Sync:** Copy once, sync everywhere. Powered by WebRTC for direct, high-speed data transfer.
- **🛡️ Secure & Private:** AES-256 end-to-end encryption ensures your clipboard data stays between *your* devices.
- **☁️ Zero Cloud:** No external servers store your data. It's a true zero-knowledge architecture.
- **📱 Quick-Paste Overlay:** Access your clipboard history instantly via a Windows hotkey or Android Quick Settings Tile.
- **📌 Smart Pinning:** Keep your most important snippets always at hand with a local SQLite vault.
- **🌐 Foreign Network Support:** Works across different networks using secure MQTT signaling.
- **🔋 Battery Optimized:** Designed to run in the background with near-zero battery impact.

## 📥 Getting Started

### Windows
1. Download the latest `.exe` from the [Releases](https://github.com/RashikFarhan/ClipSync/releases) page.
2. Run the installer and grant necessary permissions.
3. Use `Win + Shift + V` (default) to trigger the Quick-Paste menu.

### Android
1. Download the latest `.apk` from the [Releases](https://github.com/RashikFarhan/ClipSync/releases) page.
2. Install the app and enable the **Accessibility Service** for clipboard monitoring.
3. Use the **Quick Settings Tile** for instant access.

## 🛠️ Technology Stack

- **Flutter:** Cross-platform mobile and desktop core.
- **Rust/Go:** High-performance background services.
- **WebRTC:** Secure P2P data streaming.
- **SQLite:** Lightweight local persistence.
- **MQTT:** Reliable device discovery and signaling.
